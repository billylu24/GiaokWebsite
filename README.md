# Personal Website Template

A lightweight static personal website template. It includes:

- Responsive single-page layout
- Hero section with a profile visual
- Project cards
- Experience timeline
- Contact links
- Light and dark theme toggle

## Customize

Edit the placeholder content in `index.html`:

- `Your Name`
- Intro headline and bio
- Project titles and links
- Experience timeline
- Email, GitHub, and LinkedIn links

Replace `assets/profile-placeholder.svg` with your own image when ready.

## Run locally

Open `index.html` in your browser. No build step is required.

## Deploy

This template works with GitHub Pages. In the repository settings, enable Pages
from the default branch and root folder.
